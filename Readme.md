# WhisperJS

## Installation

npm:

    $ npm install whisperjs

## Usage

 To get started simply `require('whisperjs)`

## Running Tests

The test is built in vowsjs (http://vowsjs.org/). First make sure you have vow installed:

    $ npm install vows -g

Then run the tests:

    $ vows test.js
